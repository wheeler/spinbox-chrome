How to get your SoundCloud credentials

  1. Open SoundCloud in Chrome
  2. Press F12 to open Developer Tools
  3. Go to the Network tab
  4. Click the filter icon and select "Fetch/XHR"
  5. Refresh the page
  6. Look for any request to api-v2.soundcloud.com
  7. In the request headers, find:
    • Authorization: OAuth [your-token]
    • client_id: [your-client-id]
Note: These credentials are only used within your browser to access the SoundCloud API.
