# Changelog

## [0.0.1] - 2026-02-27 

Initial release version

### Added

- Ability to hide tracks on the Soundcloud Feed page
- Ability to undo recently hidden tracks in the Feed page sidebar
- Automatically play the next track when a currently playing track is hidden
- Option to disable the expand-and-contract behavior of tracks with "visuals" (background image
- Ability to export / import hidden tracks in the extension settings popup
